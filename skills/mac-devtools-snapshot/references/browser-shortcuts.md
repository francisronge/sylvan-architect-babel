# Browser DevTools Shortcuts (macOS)

- Chrome / Brave / Edge / Arc: `Cmd + Option + I`
- Safari: `Cmd + Option + I` (Develop menu enabled)

Common pane shortcuts:

- Console (Chrome-family): `Cmd + Option + J`
- Elements (Chrome-family): `Cmd + Option + C`

If a shortcut does not work, open DevTools from the browser menu.
